# EXPERIMENT2.md — 실험 라운드 2 설계 (배치 재검증 포함)

> 목적: ① "5m vs 문밀착+차폐" 배치 결정 ② v2 앙상블 실측 검증 ③ 파라미터 실측 튜닝
> 원칙: 조건당 최소 30회 통과 (기존 이격 실험은 회수 부족으로 재검증 필요)

## 사전 조건 (필수)
- [ ] 펌웨어 `scan_window = scan_interval` 적용 (미적용 시 결과 전체 무효 — 손실 40%가 모든 걸 가림)
- [ ] TX -12dBm, 광고 500ms
- [ ] GT: 스톱워치 랩 + 통과마다 방향 기록

## 실험 매트릭스

### 조건 C1 — 복도 수평 5m (현행 배치)
### 조건 C2 — 문 양쪽 밀착 (간격 = 문폭+α ≈ 1~1.5m) + 뒷면 알루미늄 포일 차폐*
*TitanRF 패브릭 대용으로 두꺼운 알루미늄 포일 3겹이면 실험용 충분 (수천 원). 안쪽 수신기는 컵 모양으로 감싸고 바깥은 뒤판만.

각 조건에서 동일 프로토콜:

| 세션 | 내용 | 횟수 |
|------|------|------|
| S1 | 보행 진입/진출 교대 | 20 |
| S2 | 구보 진입/진출 교대 | 20 |
| S3 | 전력질주 진입/진출 | 10 |
| S4 | 교차점(문앞) 정지 60초 ×3회 | FP만 측정 |
| S5 | 문앞 서성임 15초 후 진입 ×5 | 5 |
| S6 | 2인 밀착 동시 진입→동시 진출 ×5 | 20 (2인×10) |
| 합계 | | 통과 75회/조건 |

## 분석 절차
1. raw CSV 확보 (A/B 페어) → `data/raw/`에 원본 보존
2. `python3 replay.py A.csv B.csv --gt gt.csv --method peak` / `diff` / `ensemble` → 3종 × 2배치 = 6개 결과표
3. 판정 기준: 감지율 ≥95% & FP ≤2 & 전력질주 감지율 ≥80% 를 만족하는 (배치, 방법) 채택
4. 동률이면 설치 편의성으로 결정 (문밀착이 설치 표준화에 유리 — 납품 관점)

## 통계 유의성 메모
- 통과 75회 기준 감지율 95% 측정 시 95% 신뢰구간 ≈ ±5%p → 두 조건 차이가 5%p 이내면 "동등"으로 판정하고 설치 편의로 결정
- 세션 순서는 조건 간 교차 (시간대 편향 방지)
